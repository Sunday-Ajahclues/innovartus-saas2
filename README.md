# Innovartus Cloud Dashboard v2.0

## CI/CD Demonstration
1. Edit `index.html`
2. Commit:
```bash
git add .
git commit -m "Update to v2.0"
git push origin main
```
3. Render/Vercel automatically redeploys.
4. Verify the live site updates.
