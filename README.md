A modern, responsive SaaS dashboard built using **HTML, CSS, and JavaScript** to demonstrate **Cloud Computing**, **CI/CD**, **Deployment**, **Monitoring**, and **Scalability** concepts.

This project was created as part of the **Innovartus (SaaS & DevOps) Cloud Computing Case Study**.
